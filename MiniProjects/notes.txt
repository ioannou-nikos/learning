Θα προσπαθήσω να κάνω ένα framework που να μπορεί να παρακολουθεί τις δουλειές μου και γενικότερα τις εργασίες του
γραφείου. Το όλο έργο θα το ονομάσω MiniProjects και βασίζεται στην πρακάτω λογική

1. Κάθε μία δουλειά θα ονομάζεται Task. Αυτό θα περιλαμβάνει κάθε είδους εργασία που πρέπει να διαχειριστεί το γραφείο.
   Αυτό θα είναι και το βασικό αντικείμενο.

2. Κάθε Task θα έχει έναν ή μερισσότερους στόχους Goal που θα πρέπει να οδηγούνται σε κάποιο αποτέλεσμα επιτυχίας,
   αποτυχίας ή προσέγγισης (Ενδεχομένως να δίνεται και η δυνατότητα για ποσοστιαία επίτευξη με 1 την απόλυτη επιτυχία
   και 0 την απόλυτη αποτυχία)

3. Κάθε Task θα μπορεί να συνδυαστεί με άλλα και να σχηματίζει μια ομάδα TaskGroup προκειμένου να μπορούμε να
   παρακολουθούμε πιο σύνθετα έργα ή δραστηριότητες.

4. Πράγματα που συμβαίνουν και επιρρεάζουν κάποιο Task ή Goal θα ονομάζονται Events

5. Ενέργειες που θα πρέπει να γίνουν θα ονομάζονται Obligation ενώ αυτές που γίνονται θα ονομάζονται Actions. Όταν ένα
   Obligation εκτελείται θα πρέπει να μετατρέπεται αυτόματα σε Action.

6. Τα πάντα περιστρέφονται γύρω από άτομα. Άρα αντικείμενο κμβικής σημασίας είναι το άτομο με όλες του τις ιδιότητες. Το
   άτομο μπορεί να είναι ο ενδιαφερόμενος, ο υπόχρεος ενός Obligation, ο ενεργών ενός Action, ο παρεμβαίνων KeyPlayer,
   ο μεσολαβών (mediator)

7. Τα πάντα αφορούν δομές Structures. Αυτές είναι οι οργανοτικές δομές της περιφέρειας, του δημοσίου, εταιρίες,
   οργανισμοί, σύλλογοι κτλ
